Todas las anotaciones usadas en el proyecto · Significado y ubicación exacta

🏗️ Infraestructura — Capa Web

@RestController Infra

Combina @Controller + @ResponseBody. Le dice a Spring que esta clase atiende peticiones HTTP y que cada método devuelve JSON directamente (sin templates HTML).

Usado en: CourseController.java
@RequestMapping Infra

Define la ruta base para todos los endpoints de la clase. Todos los métodos del controller heredan este prefijo. Aquí vale /api/courses.

Usado en: CourseController.java
@GetMapping Infra

Ataje de @RequestMapping(method = GET). Mapea peticiones HTTP GET a un método específico. Se usa tanto en /api/courses (lista) como en /api/courses/{id}.

Usado en: CourseController.java
@RequestParam Infra

Extrae parámetros de la URL (?search=python&category=CLOUD). Con required = false el parámetro es opcional y llega null si no se envía.

Usado en: CourseController.java → getCourses()
@PathVariable Infra

Captura un segmento dinámico de la URL: /api/courses/{id} → el valor de id se inyecta directamente en el parámetro del método.

Usado en: CourseController.java → getCourseById()

⚙️ Configuración

@SpringBootApplication Config

Anotación maestra. Combina: @Configuration + @EnableAutoConfiguration + @ComponentScan. Arranca todo el contexto de Spring con una sola línea.

Usado en: CatalogApplication.java
@Configuration Config

Marca una clase como fuente de definición de beans. Los métodos anotados con @Bean dentro de ella crean objetos gestionados por Spring.

Usado en: BeanConfiguration.java · CorsConfig.java
@Bean Config

Le dice a Spring: "el objeto que este método retorna lo quiero en el contenedor de IoC". Así registramos CourseRepositoryPort y CourseUseCase manualmente, sin @Service en el dominio.

Usado en: BeanConfiguration.java

🔧 Lombok — Reducción de Boilerplate

@Getter Lombok

Genera automáticamente los métodos getX() para todos los campos de la clase. Evita escribir decenas de getters a mano.

Usado en: Course.java · CourseResponse.java
@Builder Lombok

Genera el patrón Builder. Permite crear objetos así: Course.builder().id(1L).title("Python").build(). Ideal para objetos inmutables con muchos campos.

Usado en: Course.java · CourseResponse.java
@AllArgsConstructor Lombok

Genera un constructor con todos los campos como parámetros. Necesario cuando usas @Builder con campos final para que Lombok pueda construir el objeto.

Usado en: Course.java
@RequiredArgsConstructor Lombok

Genera un constructor solo con los campos final y los marcados con @NonNull. Spring lo usa para inyección por constructor (la forma más recomendada, sin @Autowired).

Usado en: CourseService.java · CourseController.java
